自采集壁纸网站源码
========
特色：壁纸种类多、自动更新，自动采集、支持多分辨率、瀑布流加载、全站 ajax!

壁纸采集自 360壁纸库、必应首页的每日图片以及金山词霸开放平台

相关原理见 [[分享一枚获取高清壁纸的api](https://blog.a-my.top/679.html) 和 [使用PHP抓取Bing每日图像并为己所用](https://blog.a-my.top/681.html)
